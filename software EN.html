Software Development

What is Software Development?

Software development is the process of designing, creating, testing, and maintaining applications, frameworks, or other software components. It involves the application of programming languages and engineering principles to solve specific problems or improve processes in various fields such as trade, medicine, education, among others.

Technical Training

In this high school program, you will receive the technical training necessary to develop desktop, intranet, web, and mobile applications. You will also gain the necessary knowledge for managing relational databases.

Similarly, you will acquire basic skills in the area of networks, as you will learn to create, manage, and configure local area networks, configure network devices, and set up basic services on virtual machines. In addition, you will obtain skills in preventive and corrective maintenance of computers, allowing you to assemble and/or replace computer parts.

Technical Competencies

  • Analyze client requirements to develop systems.
  • Develop programming logic.
  • Design IT solutions and their components using the Unified Modeling Language (UML).
  • Manage relational databases using DBMS such as Workbench, phpMyAdmin, PostgreSQL, HeidiSQL, SQL Server Express.
  • Develop web applications under client-server architecture using programming languages in high demand in the market, such as Python, Java, C#, and frameworks like Angular, React using JavaScript or TypeScript.
  • Design and configure a virtual network and its devices using Packet Tracer, applying the most well-known network topologies.
  • Prepare user manuals using quality standards aligned with ISO and ITIL norms.
  • Implement best practices in software development.
  • Competencies in emerging technologies (Artificial Intelligence, Design and manipulation of virtual reality, 3D printing, video mapping).

Job Fields

  • Support for application maintenance.
  • Training for software development.
  • Testing and quality control of systems.
  • Cloud Computing (Development of applications that run from the Web).
  • Development of web components.
  • Tools for generating code for IT software solutions.
  • Quality assurance technician.
  • Web site and application developer.
  • System designer.
  • Database designer and administrator.
  • Installer and configurator of LAN networks.
  • Computer systems operator.
  • System documentation.
  • Sales of software solutions.
